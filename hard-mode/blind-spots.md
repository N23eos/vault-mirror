# Hard Mode — Blind Spots

> **Opt-in module.** Deliberately uncomfortable. Confirm with the user before
> running. This is analysis, not therapy — if the material gets heavy, the
> right move is a human professional, and the report should say so.

## Goal

Find what the person systematically cannot see about themselves: recurring
self-contradictions, stories they tell that the evidence undercuts, and
patterns visible to a reader but apparently invisible to the writer.

## Method

- Hunt contradictions across time: note from year X asserting A, behavior or
  later note showing not-A — especially when the assertion is repeated after
  the counter-evidence.
- Hunt self-narratives: phrases the person uses to explain themselves ("I'm
  the kind of person who...", "my problem is..."). Test each against the
  actual record. Which narratives are load-bearing but false?
- Hunt asymmetries: standards applied to others but not self (and reverse),
  topics always approached through the same defensive frame, credit/blame
  attribution patterns.
- Hunt the missing: what a person with this life would normally write about
  but this vault never mentions.
- Steelman first: before declaring a blind spot, write the strongest case
  that the person's self-story is actually right. Only what survives goes in.

## Output contract — `Analysis/blind-spots.md`

```
# Blind Spots
## Self-narratives vs record   (each: the story, the evidence, the verdict)
## Contradictions              (assertion vs behavior, with dates)
## Asymmetries                 (double standards, framing habits)
## The unmentioned             (conspicuous absences, marked hypothesis)
## Steelman appendix           (the cases where the self-story won)
```

## Rules reminder

Apply the global rules from `prompts/00-start.md`. The honesty rule is the
point of this module; the kindness half of it still applies.

## Quality bar

- Did every blind spot survive its steelman?
- Is each contradiction dated and linked on both sides?
- Would this read as precise rather than cruel? Precision is the license for
  this module's existence.
